Bottom line — June 2026

The system is stable and trading is running — but three critical items need a decision within 30 days.

  • Production memory is maxed out — all 3 trading hosts at 90–94% RAM. One host failure could take trading offline.
  • ~54 servers run end-of-life CentOS 7 (no security patches since June 2024).
  • The UAT/test platform is unsupported (VMware ESXi 6.0, end of support since 2020).
  • No disaster-recovery site — a single data-center event would halt trading completely.
